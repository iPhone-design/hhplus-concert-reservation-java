// 토큰 발급 및 대기열 등록
유저->시스템: 토큰 발급 요청
시스템->유저DB: 이용자 확인 요청
유저DB->시스템: 이용자 확인 응답
시스템-->유저: 이용자 확인 실패 시 응답
시스템->토큰DB: 토큰 저장 요청
토큰DB->시스템: 토큰 저장 응답
시스템->대기열DB: 대기열 추가 요청
대기열DB->시스템: 대기열 추가 응답
시스템->유저: 토큰 발급 응답

// 좌석 예약
유저->시스템: 좌석 예약 요청
시스템->토큰DB: 토큰 확인 요청
토큰DB->시스템: 토큰 확인 응답
시스템-->유저: 토큰 확인 실패 시 응답
시스템->대기열DB: 대기열 순서 확인 (폴링)
대기열DB->시스템: 대기열 순서 응답
시스템->좌석DB: 좌석 상태 값 변경 요청 (임시 배정)
좌석DB->시스템: 좌석 상태 값 변경 응답 (임시 배정)
시스템->대기열DB: 대기열 삭제 요청
대기열DB->시스템: 대기열 삭제 응답
시스템->유저: 좌석 임시 배정 응답

// 결제
유저->시스템: 결제 요청
시스템->토큰DB: 토큰 확인 요청
토큰DB->시스템: 토큰 확인 응답
시스템-->유저: 토큰 확인 실패 시 응답
시스템->결제DB: 결제 요청
결제DB->시스템: 결제 응답
시스템-->유저: 잔액 부족 시 실패 응답
시스템->예약DB: 예약 요청
예약DB->시스템: 예약 응답
시스템->좌석DB: 좌석 상태 값 변경 요청 (예약)
좌석DB->시스템: 좌석 상태 값 변경 응답 (예약)
시스템->토큰DB: 토큰 만료 요청
토큰DB->시스템: 토큰 만료 응답
시스템->유저: 결제 응답
